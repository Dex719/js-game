# Maze Escape

Браузерная 2D-игра — лабиринт с врагами, ловушками и ключами. Каждый уровень генерируется процедурно, поэтому одинаковых прохождений не бывает.

## О проекте

Сделал чтобы разобраться как работает игровой движок Phaser 3 — игровой цикл, коллизии, сцены и работа с состоянием. Параллельно практиковал разбивку кода на классы: генератор лабиринта, менеджер уровней и UI живут отдельно.

## Стек

- **Phaser 3.70** — игровой движок
- **Vanilla JS** — логика игры (ES6 классы)
- **localStorage** — сохранение прогресса и звёзд

## Геймплей

- 10 уровней с нарастающей сложностью
- Процедурная генерация лабиринта (алгоритм recursive backtracking)
- Враги, шипы, фальшивые полы, замки и ключи
- Система звёзд (до 3-х) за скорость и здоровье
- Неуязвимость после получения урона (мигание)
- Тряска камеры и красная вспышка при ударе

## Управление

| Кнопка | Действие |
|---|---|
| `WASD` / `↑↓←→` | Движение |
| `Shift` | Бег |
| `E` | Взаимодействие (открыть дверь) |
| `Space` | Прыжок |
| `ESC` | Пауза |

## Запуск

Открой `index.html` в браузере — Phaser подключается через CDN, установка не нужна.

Или через локальный сервер:

```bash
python -m http.server 8080
```

Затем открыть **http://localhost:8080**

## Архитектура

```
js/
├── main.js       — конфиг Phaser и запуск игры
├── game.js       — главная сцена, игровой цикл
├── mazegen.js    — генерация лабиринта, расстановка врагов и ловушек
├── levels.js     — данные уровней, прогресс (localStorage)
└── ui.js         — меню, HUD, экраны победы/поражения
```
